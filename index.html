    Following the CPIC template: l, r are inclusive bounds on the unsearched range, and the loop runs while l <= r.     Unsearched [l, r]
    mid (testing)
    True "good" region
    ans (best so far)
  


  

    The BSTA template: the answer space is a range [lo, hi]. check(x) tests whether x is feasible. Because the feasibility is monotonic (if x works, every value on the better side of x also works), one binary search finds the boundary. When check(mid) is true, record ans = mid and shrink toward the better extreme; otherwise shrink the other way.
    


    Keyboard: → or Space step · ← back · R reset.
